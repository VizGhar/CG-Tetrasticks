The Goal

Little Jimmy had just solved some puzzle, but while he went for a champa... I mean milk... disaster struck — the game board fell to the floor! And now, only a few pieces remain correctly placed, and the rest are scattered.

You must quickly restore the puzzle before Jimmy notices, or he'll throw a fit again! Carefully place the missing pieces to complete the shape, making sure everything fits perfectly.

  Rules

You will be given a board description, that have to be entirely covered using the available tetrasticks.
  • tetrasticks can be flipped horizontally
  • and/or rotated by 0°, 90°, 180°, or 270°
  • below is an overview of all 16 tetrasticks used in the puzzle with their respective IDs, partially resembling their shapes

Symbol o marks space that can be shared by two tetrasticks, while symbol O marks space that cannot be shared with other tetrasticks. See shared spaces highlighted by red circles in image below:

Highlighted shared parts between pairs of tetrasticks

Your task is to fill board of size 5 x 5 squares or better 6 x 6 dividers with provided tetrasticks. Some tetrasticks can already be placed on board. So pay attention to your input.

Every turn you have to place single tetrastick on board. You have to pick it by providing its id, whether you want to flip it horizontally or not, how many times you want to rotate the piece clockwise and where you want to place it, by providing row and column

  Example

If you want to place J-shaped tetrastick on board as shown in example above, you have to rotate it once and place it on position 0, 0 therefor your turn output should be exactly:
J 0 1 0 0

Placing Z shaped tetrastick would require horizontal flip and position 2, 2, so output should look like this:
Z 1 0 2 2

  Game Input

Input

Every turn you will receive current configuration in following format:

Line 1: number n representing amount of remaining tetrasticks to place

Line 2: n space separated IDs of remaining tetrasticks

Line 3: space separated h - height and w - width of game board

Following lines: h strings of length w representing game board, where

  • . - can be occupied
  • space - can't be occupied - center of square
  • / - can't be occupied - already occupied by two pieces
  • x (lowercase) - occupied by tetrastick with id = x, but can still can host one more
  • X - already occupied by tetrastick with id = X

Output

1 Line: 5 Space separated values id, flip, rotations, row, column

id of tetrastick
flip 1 if tetrastick should be flipped horizontally before placing on board 0 otherwise
rotations from interval <0 ; 3> - count of clockwise rotations
row from interval <0 ; 5> - vertical placement
column from interval <0 ; 5> - horizontal placement
For example H 1 2 0 3 will flip (1) horizontally h - shaped tetrastick (H) and rotate it 2 times clockwise. After that the tetrastick will be placed to first row and fourth column.
Constraints

Allotted response time to output is ≤ 2s in first turn and 50ms in remaining turns
